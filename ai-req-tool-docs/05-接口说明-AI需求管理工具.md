# 接口说明：AI需求管理工具（严格对齐 DocGenController）

统一响应包装：

```json
{
  "code": 0,
  "message": "ok",
  "data": {},
  "traceId": "uuid"
}
```

请求头：
- `X-Trace-Id`：可选，不传时后端自动生成。

## 1. 健康检查
- `GET /api/docgen/health`

`data`:

```json
{
  "status": "UP"
}
```

## 2. 创建任务
- `POST /api/docgen/jobs`

请求体 `CreateJobRequest`:

```json
{
  "businessDescription": "string, max=20000",
  "previousPrdMarkdown": "string, optional, max=200000"
}
```

响应体 `CreateJobResponse`:

```json
{
  "jobId": "string",
  "status": "PENDING|CLARIFYING|READY|COMPLETED",
  "clarifyQuestions": [],
  "prdMarkdown": "string|null"
}
```

## 3. 查询任务
- `GET /api/docgen/jobs/{jobId}`

响应同 `CreateJobResponse`。

## 4. 提交澄清（兼容接口）
- `POST /api/docgen/jobs/{jobId}/clarify`

请求体：

```json
{
  "key1": "value1",
  "key2": "value2"
}
```

响应同 `CreateJobResponse`。

## 5. 聊天澄清
- `POST /api/docgen/jobs/{jobId}/chat`

请求体 `ChatRequest`:

```json
{
  "message": "string, max=4000"
}
```

响应体 `ChatResponse`:

```json
{
  "jobId": "string",
  "status": "PENDING|CLARIFYING|READY|COMPLETED",
  "assistantMessage": "string",
  "readyToGenerate": true,
  "pendingQuestion": "string",
  "confirmedItems": ["string"],
  "unconfirmedItems": ["string"],
  "chatHistory": [
    {
      "role": "user|assistant",
      "content": "string"
    }
  ],
  "prdMarkdown": "string|null",
  "basePrdMarkdown": "string|null"
}
```

## 6. 生成 PRD
- `POST /api/docgen/jobs/{jobId}/generate`

响应体同 `ChatResponse`。

## 7. 导出 PRD
- `GET /api/docgen/jobs/{jobId}/export`

响应：
- `Content-Type: text/markdown;charset=UTF-8`
- `Content-Disposition: attachment; filename="PRD-{jobId}-revised.md"`
- Body: markdown bytes

## 8. 错误响应
- 业务错误：`code=10001`
- 系统错误：`code=20001`
