# 功能清单：AI需求管理工具（可直接拆开发任务）

## 1. 已实现能力（基于现有代码）
- 任务创建：输入需求描述与旧版 PRD。
- 任务查询：根据 `jobId` 查询任务状态。
- AI 澄清会话：多轮问答，更新确认清单。
- 条件生成：待确认项清空后可生成 PRD。
- 文档导出：下载 Markdown 文件。

## 2. 待开发任务（后端）
- 增加任务列表与分页查询接口。
- 增加任务删除/归档接口。
- 增加 `chat` 防重复提交与幂等键。
- 增加版本快照接口：
  - 保存版本
  - 查询版本列表
  - 查询版本差异

## 3. 待开发任务（前端）
- 增加任务列表页（按状态筛选）。
- 增加版本对比页（逐段 diff）。
- 增加生成结果质量检查提示（缺失章节提醒）。
- 增加任务备注与标签管理。

## 4. 待开发任务（通用）
- 鉴权与权限控制。
- 审计日志入库。
- 错误码标准化与可观测看板。
