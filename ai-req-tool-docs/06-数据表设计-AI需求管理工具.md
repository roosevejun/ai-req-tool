# 数据表设计：AI需求管理工具（按当前字段映射）

## 1. docgen_job
用途：映射 `JobState` 核心字段。

| 字段 | 类型 | 说明 |
|---|---|---|
| id | bigint PK | 主键 |
| job_id | varchar(64) UNIQUE | 对外任务 ID |
| business_description | longtext | `businessDescription` |
| status | varchar(32) | `PENDING/CLARIFYING/READY/COMPLETED` |
| ready_to_generate | tinyint | `readyToGenerate` |
| pending_question | varchar(1000) | `pendingQuestion` |
| prd_markdown | longtext | `prdMarkdown` |
| base_prd_markdown | longtext | `basePrdMarkdown` |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |

## 2. docgen_chat_history
用途：映射 `chatHistory`。

| 字段 | 类型 | 说明 |
|---|---|---|
| id | bigint PK | 主键 |
| job_id | varchar(64) INDEX | 任务 ID |
| role | varchar(16) | `user/assistant` |
| content | longtext | 消息内容 |
| seq_no | int | 顺序号 |
| created_at | datetime | 创建时间 |

## 3. docgen_confirm_item
用途：映射 `confirmedItems` / `unconfirmedItems`。

| 字段 | 类型 | 说明 |
|---|---|---|
| id | bigint PK | 主键 |
| job_id | varchar(64) INDEX | 任务 ID |
| item_text | varchar(1000) | 条目内容 |
| item_status | varchar(16) | `CONFIRMED/UNCONFIRMED` |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |

## 4. docgen_answer_map
用途：映射 `answers(Map<String,String>)`。

| 字段 | 类型 | 说明 |
|---|---|---|
| id | bigint PK | 主键 |
| job_id | varchar(64) INDEX | 任务 ID |
| answer_key | varchar(255) | key |
| answer_value | longtext | value |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |

## 5. docgen_version_snapshot（扩展建议）
用途：补齐版本管理能力。

| 字段 | 类型 | 说明 |
|---|---|---|
| id | bigint PK | 主键 |
| job_id | varchar(64) INDEX | 任务 ID |
| version_no | int | 版本号 |
| prd_markdown | longtext | 快照内容 |
| change_summary | longtext | 差异摘要 |
| created_at | datetime | 创建时间 |
